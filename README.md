## `>` Tech Behind This Site

Built entirely with **vanilla HTML, CSS & JavaScript** — no frameworks, no
build step. Features a matrix-rain canvas background, glassmorphic sticky
navbar, scroll-triggered reveals, animated skill bars, a terminal-style
boot sequence, and full responsive support down to mobile.

```
cybersantanu.github.io/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
    ├── profile.jpg
    ├── resume.pdf
    └── certificates/
```

To run locally, simply open `index.html` in a browser — or serve the folder
with any static file server. To deploy, push to a repo named
`cybersantanu.github.io` and enable GitHub Pages.
